# DoNote — 纯本地 Windows 日程管理软件

**DoNote** 是一款 Windows 桌面日程管理工具，纯本地运行，无需联网，无需注册。用自然语言一句话就能创建日程，支持今日/周/月/年多种视图，让你的时间安排一目了然。

## 主要功能

### 智能输入

在顶部输入框用自然语言输入，软件会自动识别日期和时间。例如：

- `明天下午3点开会`
- `下周五之前提交报告`
- `每天晚上8点健身`

基于 chrono-node 中文语义解析，无需手动选择日期。

### 多视图切换

| 视图 | 说明 |
|---|---|
| 今日 | 时间线展示今天所有任务，已完成任务自动下沉 |
| 本周 | 按周一到周日分组展示 |
| 未来7天 | 快速预览接下来一周的日程 |
| 本月 | 日历网格视图，点击任意日期查看或添加任务 |
| 全年 | 4×3 网格全年概览，日期右下角彩色圆点标记当天事件，鼠标悬浮查看详情 |
| 已完成 | 集中查看所有已完成任务 |

### 任务管理

- 每个任务支持**标题、备注、分类、优先级、日期时间**
- 6 个内置分类：工作（蓝）、会议（紫）、生活（绿）、学习（黄）、健康（红）、其他（灰）
- 支持**每日重复**：设置后自动在每一天生成任务副本，取消时自动清理
- **逾期管理**：过期任务明确标记，可确认或延期
- **延期功能**：一键将未完成任务推迟到未来某天
- 键盘快捷键操作（年视图支持 ←/→ 切换年份、Space 回今日）

### 数据安全

- 所有数据保存在**本地 JSON 文件**，不经过任何服务器
- 可在设置中**自定义存储目录**
- 每次保存**自动备份**到 backup 子目录，文件名带时间戳，随时可恢复
- 支持**数据迁移**：更换存储路径时自动迁移已有数据

### 交互体验

- 无边框窗口设计，界面简洁
- 关闭窗口后**最小化到系统托盘**，后台运行
- 系统原生通知提醒
- 复选框单击完成，点击卡片编辑，操作流畅

## 技术栈

| 技术 | 用途 |
|---|---|
| **Electron 28** | 桌面应用框架，窗口管理、系统托盘、原生通知、文件 IO |
| **React 18** | 前端 UI 框架 |
| **TypeScript 5.4** | 类型安全开发 |
| **Vite 5** | 构建工具，极速热更新 |
| **electron-builder** | 打包为 NSIS 安装包 / 便携版 exe |
| **chrono-node 2.7** | 自然语言日期时间解析（支持中文） |
| **uuid** | 任务唯一 ID 生成 |

### 架构概览

```
Electron Main Process    → 窗口管理、系统托盘、文件读写、通知
        ↕ IPC (contextBridge)
React Renderer Process   → 全部 UI 组件、状态管理、用户交互
        ↕
本地文件系统             → tasks.json + backup/ 自动备份
```
